Student Dropout Prediction and Academic Success

Project Description

This project aims to develop a machine learning model that predicts whether a student is likely to graduate, enroll, or drop out based on demographic, academic, and socioeconomic information. The project applies data preprocessing, exploratory data analysis, model training, and performance evaluation using supervised machine learning algorithms.

Goal / Objectives

- Predict students at risk of dropping out, enrolled and graduate.
- Compare multiple machine learning algorithms.
- Determine the best-performing model.
- Help educational institutions identify at-risk students for early intervention.

Dataset Name:

Predict Students Dropout and Academic Success Dataset

Target Variable
Target

Possible values:

- Graduate
- Enrolled
- Drop
